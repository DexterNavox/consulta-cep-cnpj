Informações sobre a API ViaCEP

O ViaCEP é um webservice gratuito para consulta de Código de Endereçamento Postal (CEP) do Brasil.

  • Formato do CEP: 8 dígitos (ex: 01001000)
  • Formato de retorno: JSON ou XML
  • CEP inválido: retorna código 400 (Bad Request)
  • CEP válido mas inexistente: retorna objeto com "erro": true

Para mais informações, visite: viacep.com.br
