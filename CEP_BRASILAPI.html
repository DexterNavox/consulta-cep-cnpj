Informações sobre a BrasilAPI v1

A BrasilAPI é um projeto de código aberto que oferece uma variedade de endpoints para consulta de dados públicos do Brasil.

CEP v1 (Formato clássico):
  • Formato do CEP: 8 dígitos (ex: 01001000)
  • Formato similar ao ViaCEP
  • Retorna dados completos do endereço
  • CEP inválido/não encontrado: retorna código 400 (Bad Request)

Para mais informações, visite: brasilapi.com.br
