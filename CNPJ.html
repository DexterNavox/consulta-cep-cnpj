Informações sobre a API CNPJA

A API Pública CNPJA permite consultar informações de empresas brasileiras de forma gratuita e sem necessidade de cadastro.

  • Limite: 5 consultas por minuto por IP
  • Atualização: Dados atualizados a cada 45 dias para empresas ativas
  • Fontes: Receita Federal, Simples Nacional, Cadastro de Contribuintes, SUFRAMA
  • Formato: JSON
Atenção: Os dados podem apresentar defasagem em relação à fonte original. Para dados em tempo real e funcionalidades adicionais, considere a API Comercial.

Para mais informações, visite: open.cnpja.com
